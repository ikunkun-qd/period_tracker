# 女性生理期记录APP详细需求文档

## 1. 项目概述

### 1.1 产品定位
打造一款功能全面、界面友好、数据准确的女性生理健康管理应用，帮助用户科学地记录、预测和管理生理周期，提升女性健康意识。

### 1.2 目标用户
- 主要用户：18-45岁有生理期的女性
- 次要用户：备孕女性、新手妈妈、关注女性健康的用户

### 1.3 核心价值
- 精准预测生理周期和排卵期
- 全面的健康数据记录与分析
- 个性化的健康建议和提醒
- 保护用户隐私和数据安全

## 2. 核心功能模块详细说明

### 2.1 生理周期管理

#### 2.1.1 经期记录功能
**基础记录模块**
- **经期开始/结束标记**
  - 一键标记经期开始，自动记录时间戳
  - 手动调整开始时间（支持回溯3天内修改）
  - 经期结束确认，自动计算持续天数
  - 支持异常情况记录（提前结束、延长等）
  - 历史记录修改功能，保留修改日志

- **经期持续天数统计**
  - 自动计算当次经期天数
  - 显示近6个月平均持续天数
  - 异常天数预警（少于3天或超过8天）
  - 持续天数趋势图表展示

- **月经流量等级记录**
  - 5级流量分类：点滴、轻微、正常、偏重、很重
  - 每日流量记录，支持早中晚三个时段
  - 卫生用品使用量记录（卫生巾、棉条数量）
  - 流量变化曲线图显示
  - 异常流量预警（连续重度流量或突然停止）

- **经期颜色和质地记录**
  - 7种颜色分类：鲜红、暗红、褐色、粉红、橙色、灰色、黑色
  - 质地记录：液体、凝块、纤维状
  - 异味记录选项
  - 颜色变化时间轴显示
  - 异常颜色健康提醒

**症状记录模块**
- **疼痛管理系统**
  - 10级疼痛评分滑块
  - 疼痛部位身体图选择：下腹部、腰部、背部、头部、乳房、大腿
  - 疼痛类型：钝痛、刺痛、痉挛、胀痛
  - 疼痛持续时间记录
  - 缓解方法记录（药物、热敷、按摩等）
  - 疼痛强度时间曲线图

- **PMS症状全面记录**
  - **情绪症状**：易怒、焦虑、抑郁、情绪不稳、哭泣
  - **身体症状**：乳房胀痛、腹胀、水肿、头痛、背痛、关节痛
  - **行为症状**：食欲改变、睡眠问题、注意力不集中、社交回避
  - **其他症状**：疲劳、恶心、便秘、腹泻、皮肤问题
  - 症状严重程度1-5级评分
  - 症状出现时间记录（经前几天开始）

#### 2.1.2 智能周期预测
**预测算法系统**
- **基础预测模型**
  - 基于最近6个周期的移动平均算法
  - 考虑周期长度变化趋势
  - 季节性因素调整（季节、压力、生活方式变化）
  - 年龄相关的周期变化模型

- **AI智能学习预测**
  - 机器学习算法分析个人历史数据
  - 结合症状模式优化预测
  - 生活方式因素权重计算
  - 预测准确度持续学习提升

- **多维度预测信息**
  - 下次经期开始日期（±2天误差范围）
  - 经期持续天数预测
  - 排卵日预测（±1天精确度）
  - 生育窗口期计算（排卵前5天到排卵后1天）
  - 安全期计算（低受孕概率期间）

- **预测准确度管理**
  - 实时准确度百分比显示
  - 预测偏差分析报告
  - 算法自动调优机制
  - 用户反馈集成优化

#### 2.1.3 日历视图功能
**多维度日历显示**
- **月历视图**
  - 经期标记（深红色）
  - 排卵期标记（绿色）
  - 生育窗口标记（浅绿色）
  - 安全期标记（蓝色）
  - PMS期标记（黄色）
  - 预测期标记（虚线边框）

- **年历总览**
  - 12个月缩略日历
  - 周期规律性可视化
  - 异常月份高亮显示
  - 年度统计数据概览

- **自定义标记系统**
  - 重要日期标注（约会、旅行、运动比赛等）
  - 个人备注添加
  - 照片附件上传
  - 颜色标签分类管理
  - 提醒事项关联

### 2.2 健康数据记录系统

#### 2.2.1 身体指标监测
**基础生理数据管理**
- **体重管理模块**
  - 每日体重记录（支持小数点后1位）
  - 自动BMI计算显示
  - 体重变化趋势图（日、周、月、年视图）
  - 与经期关联的体重波动分析
  - 目标体重设置和进度跟踪
  - 体重异常波动预警

- **基础体温跟踪**
  - 精确到0.01°C的体温记录
  - 最佳测量时间提醒（起床后测量）
  - 基础体温曲线图显示
  - 排卵期体温变化识别
  - 发烧等异常体温标记
  - 体温计数据自动同步（蓝牙设备）

- **血压心率监测**
  - 收缩压/舒张压记录
  - 静息心率记录
  - 血压分类自动判断（正常、偏高、高血压）
  - 运动后心率恢复监测
  - 异常血压/心率预警
  - 医疗级数据导出格式

**妇科健康专项指标**
- **宫颈黏液观察记录**
  - 6种质地分类：干燥、粘稠、乳霜状、水样、蛋清样、拉丝状
  - 颜色记录：透明、白色、黄色、其他
  - 量的评估：无、少量、中等、大量
  - 排卵期黏液特征自动识别
  - 感染风险预警提示

- **乳房健康监测**
  - 乳房胀痛程度1-10级评分
  - 胀痛部位标记（左侧、右侧、双侧）
  - 乳房检查提醒（月经后一周最佳）
  - 异常肿块记录提醒
  - 乳头异常分泌物记录

- **阴道健康状况**
  - 分泌物量的评估
  - 分泌物颜色和质地记录
  - 异味程度记录
  - 瘙痒不适程度评分
  - 感染症状预警系统

#### 2.2.2 生活方式详细记录
**饮食管理系统**
- **饮水量跟踪**
  - 每日饮水目标设置（推荐1500-2000ml）
  - 快速记录按钮（250ml、500ml、750ml）
  - 饮水类型记录：纯水、茶、咖啡、果汁等
  - 饮水提醒推送（可设置间隔时间）
  - 脱水预警系统
  - 与经期相关的饮水建议

- **咖啡因摄入管理**
  - 每日咖啡因摄入量计算
  - 饮品咖啡因含量数据库（咖啡、茶、可乐、能量饮料）
  - 经期咖啡因摄入建议
  - 过量摄入预警（日摄入>400mg）
  - 戒断症状记录

- **营养补充剂管理**
  - 常用补充剂清单：叶酸、铁剂、维生素D、钙片等
  - 服用时间和剂量记录
  - 服药提醒设置
  - 补充剂与经期症状关联分析
  - 药物相互作用提醒

**运动健身跟踪**
- **运动类型和强度**
  - 运动类型分类：有氧、无氧、瑜伽、普拉提、游泳、跑步、骑行等
  - 运动强度分级：低强度、中强度、高强度、极限强度
  - 运动时长精确记录（分钟）
  - 运动频次统计（周、月频次）
  - 卡路里消耗自动计算

- **经期运动指导**
  - 不同经期阶段运动建议
  - 痛经期运动禁忌提醒
  - 经期前运动强度调整建议
  - 排卵期运动最佳时机提示
  - 运动与经期症状改善关联分析

- **运动数据同步**
  - 智能手环/手表数据自动导入
  - 第三方运动APP数据同步
  - 步数、心率、卡路里等综合数据
  - 运动目标设定和完成度追踪

**睡眠质量监测**
- **睡眠时间记录**
  - 就寝时间和起床时间
  - 实际睡眠时长计算
  - 入睡时长记录
  - 夜间醒来次数和时长
  - 睡眠效率计算（实际睡眠时间/在床时间）

- **睡眠质量评估**
  - 主观睡眠质量评分（1-10级）
  - 睡醒后精神状态记录
  - 做梦情况记录
  - 睡眠环境因素：温度、噪音、光线
  - 睡前活动记录：看书、看手机、运动等

- **睡眠与经期关联**
  - 经期不同阶段睡眠模式分析
  - 失眠与PMS症状关联分析
  - 睡眠质量对经期症状的影响评估
  - 改善睡眠的个性化建议

#### 2.2.3 情绪心理健康
**情绪状态全面记录**
- **情绪类型识别**
  - 基础情绪：快乐、悲伤、愤怒、恐惧、惊讶、厌恶
  - 复合情绪：焦虑、抑郁、烦躁、兴奋、平静、困惑
  - 情绪强度1-10级评分
  - 情绪持续时间记录

  - 压力等级1-10级自评
  - 压力相关身体症状记录
  - 压力管理技巧推荐

- **心理健康工具**

  - 焦虑自评量表（GAD-7）
  - 压力感知量表（PSS）
  - 情绪日记撰写功能
  - 心理健康趋势分析
